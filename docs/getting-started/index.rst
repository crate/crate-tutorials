============================
Getting Started With CrateDB
============================

CrateDB is a distributed SQL database that makes it simple to store and analyze
massive amounts of machine data in real-time.

This is a step-by-step tutorial that will show you how to:

- Get CrateDB up and running on your personal computer
- Import test data and how to query
- Start building your first CrateDB application
